# PlatformerMicroGame

A simple 2D platformer developed in Unity that demonstrates the implementation of core platformer mechanics and gameplay systems. Inspired by classic platformers such as Super Mario Bros., the project focuses on responsive player controls, interactive level elements, and clean game logic.

> 🚧 This project represents the first public release and serves as a learning project and portfolio piece.

## Features

- Player movement
- Double jump
- Collectibles
- Dynamic traps
- Checkpoints
- Two playable levels
- Main menu and in-game UI

## Technologies

- Unity
- C#
- Unity Physics
- Unity UI

## Gameplay

Navigate through two handcrafted levels while avoiding dynamic traps, collecting items, and reaching the finish. Use precise movement and double jumps to overcome platforming challenges inspired by classic platformer games.

## Project Goals

This project was created to practice:

- Object-Oriented Programming in C#
- Unity game development
- Player movement implementation
- Physics-based gameplay
- UI development
- Scene management
- Game architecture fundamentals
